# Smart CCraft
Make Computercraft more smart!

### Features:

  - Charge Turtles with RF
  - Rotate the Charge Station using a Hammer or a Wrench
  - Package items using 2x2 or 3x3 pattern
  - Decorative Blocks
  - WAILA Itegrated
